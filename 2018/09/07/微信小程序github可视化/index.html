微信小程序github可视化

前言

一直想做一些数据可视化的东西，所以就尝试在微信小程序中获取用户的一些GitHub上的数据以后，使用wx-charts来进行开发，GitHub的api可参考GitHubApi，目前只是获取了数据暂时展示了一下，样式请忽略😁，项目持续更新。
